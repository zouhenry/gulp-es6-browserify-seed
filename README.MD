#Angular ES6 Webpack Seed Project
## Karma, Jasmine, PhantomJS, Istanbul (Coverage) 

##Getting started

```bash
# clone this project
git clone https://github.com/zouhenry/gulp-es6-webpack-seed.git
# install packages
npm install
# build and watch using webpack dev server
npm start
# test
npm test
# test with watch
npm tdd
```
